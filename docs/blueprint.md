# **App Name**: Locked Codes

## Core Features:

- User Authentication: User authentication handled through random username generation.
- Player Panels: Display for game panels showing each player's name and a dedicated space for their guesses.
- Secret Code Input: Screen where players can enter their 4-digit secret code.
- Guessing Interface: Display to enter guesses, showing previous attempts, with digits in the correct position highlighted in green (Wordle-style feedback).
- Turn Indicator: Simple turn indicator displaying who should currently be guessing
- Exit Button: An 'Exit Game' button to end the game and initiate data removal.

## Style Guidelines:

- Primary color: #FFF44F (bright yellow)
- Background: #2E2E2E
- Font: 'Inter', a sans-serif font that offers a clean and modern look, suitable for both headers and body text, providing excellent readability across the platform.
- A clear, blocky layout. Each UI section occupies its own clearly defined region of the screen, similar to a printed form. Sections like username generation and game entry should each use separate and full screens for maximum clarity.
- Simple and clear icons that match the style, specifically a lock icon when user gets the right digit.
